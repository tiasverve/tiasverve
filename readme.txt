Чтобы запустить проект необходимо открыть папку ill1 и запустить одноименный файл. Переход к остальным иллюстрациям осуществлен при помощи стрелок на иллюстрациях.

Анимация объектов на иллюстрации осуществляется с помощью таблицы стилей CSS и библиотеки JS - JQUERY.
1.	На первой иллюстрации анимированы письмо с ключом, они начинают двигаться по клику мышки. Также кролик играет на трубе и слышна мелодия игры. Голову кота можно передвигать в любые места, и она исчезает на время перемещения.
2.	На второй иллюстрации по клику мыши королева начинает злиться, а слуга дрожать. Алиса совершает поклон королеве при наведении на неё мышью.
3.	На третьей иллюстрации можно передвигать самого крупного ежа рядом с Герцогиней, а также видно, как она кивает головой при виде Алисы.
4.	На четвертой при клике мышки чай чуть выплескивается из чашки, а также часы дрожат.
